# Instructions for `serial-communication`
1. Open the Arduino IDE.
2. Input the provided file (`arduino.ino`) into the editor.
3. Click `Tools`, then find the `Port`. Normally, this will be `COM3` or `COM4`, but it may be different.
4. Run the program.
5. Open a Python IDE, such as `PyCharm`.
6. Ensure `pyserial` is installed.
7. Input the provided file (`python.py`) into the editor.
8. Run the program.
